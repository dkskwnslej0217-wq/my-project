\# run\_v8



\## 실행 방법

이 파일을 읽으면 아래 순서로 실행해.



\## 입력

\- 주제: \[여기 입력] (비워두면 direction.md 보고 판단)

\- 플랫폼: \[Threads / Instagram / YouTube / 자동판단]



\## 실행 순서



1\. memory/direction.md 읽기 → 지금 방향 파악

2\. memory/context.md 읽기 (최근 3개만)

3\. 플랫폼 판단

4\. 해당 skills 파일 1개만 읽기

5\. direction.md의 "다음에 시도할 것" 반영해서 콘텐츠 생성

6\. output/에 저장

7\. memory/context.md 업데이트 (새 형식으로)

8\. memory/direction.md 업데이트:

&#x20;  - 이번 실행에서 뭘 시도했는지

&#x20;  - 잘된 점 / 아쉬운 점

&#x20;  - 다음 실행에서 시도할 것

9\. 완료 보고 (3줄)



\## 하지 말 것

\- direction.md 없으면 만들지 말고 context.md만 보고 판단

\- 주제 없어도 direction.md + context.md 보고 스스로 제안
