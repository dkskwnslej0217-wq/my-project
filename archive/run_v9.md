# run_v9

## 실행 방법
이 파일을 읽으면 아래 순서로 실행해.

## 입력
- 주제: [여기 입력] (비워두면 direction.md 보고 판단)
- 플랫폼: [Threads / Instagram / YouTube / 자동판단]

## 실행 순서

1. memory/direction.md 읽기 → 현재 방향 파악
2. memory/context.md 읽기 (최근 3개만)
3. 플랫폼 판단
4. brain/알고리즘.md 읽기 (해당 플랫폼 부분만 참고)
5. 해당 플랫폼 skills 파일 읽기
6. brain/패턴.md 읽기 → 훅 구조 선택
7. brain/심리.md 읽기 → 선택한 구조가 왜 먹히는지 확인
8. 콘텐츠 생성
9. output/ 저장
10. memory/context.md 업데이트
11. memory/direction.md 업데이트
12. 자기개선 판단 (같은 아쉬운 점 2회 이상이면 skills 해당 섹션 수정)
13. 완료 보고 (3줄)
