\## 실행 방법

이 파일을 읽으면 아래 순서로 실행해.



\## 입력

\- 주제: \[여기 입력]

\- 플랫폼: \[Threads / Instagram / YouTube / 자동판단]



\## 실행 순서



1\. memory/context.md 읽기 (최근 5개 항목만)

2\. 플랫폼 판단 (명시 없으면 CLAUDE.md 판단 기준 적용)

3\. 해당 플랫폼 skills 파일 1개만 읽기

4\. 콘텐츠 생성

5\. output/에 저장

6\. memory/context.md 업데이트 (한 줄)

7\. 완료 보고 (3줄 이내)
