## ⚠️ 세션 시작 시 필수 체크
1. memory/오늘요약.md 파일 열기
2. 파일 맨 위 날짜 확인
3. 오늘 날짜(실행 당일)와 다르면 → 아래 경고 출력 후 중단:
   "⚠️ 오늘요약.md가 오래됐어. 날짜: [파일날짜] / 오늘: [오늘날짜]. run_core.md 실행해서 먼저 갱신해줘."
4. 날짜 일치하면 → 정상 진행
