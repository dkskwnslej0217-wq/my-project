# 개인화 학습

## 목표
내 말투, 스타일, 잘 된 훅 패턴을 자동으로 학습해서
점점 나만의 AI로 진화

## 학습 항목
1. 훅 문장 패턴
   - output/ 폴더에서 score_hook 8점 이상 콘텐츠 분석
   - 공통 구조 추출 (반전형/숫자형/질문형 등)
   - 성공 훅 패턴 목록 자동 업데이트

2. 말투/스타일
   - 짧고 직관적인 문장
   - 구어체 위주
   - 전문용어 최소화

3. 댓글 유도 패턴
   - score_comment 8점 이상 콘텐츠 분석
   - 잘 된 댓글 유도 문장 구조 추출
   - 업데이트 주기: 10회 실행마다

## 적용 방법
콘텐츠 생성 시 이 파일 읽고
성공 패턴 우선 적용
