# Make.com 자동 업로드

## 목표
콘텐츠 생성 완료 시 Make.com으로
유튜브/인스타 자동 업로드

## Make.com 시나리오 구조

### 시나리오 1 — 유튜브 쇼츠
트리거: Webhook (Claude Code에서 호출)
액션 1: Google Drive에 영상 파일 저장
액션 2: YouTube API → 업로드 예약
액션 3: 구글시트에 업로드 기록

### 시나리오 2 — 인스타 릴스
트리거: Webhook
액션 1: Instagram Graph API → 업로드
액션 2: 구글시트에 업로드 기록

## 업로드 제한 규칙
- 유튜브: 하루 최대 5개
- 인스타: 하루 최대 3개
- 새벽 2시~6시 업로드 금지
- 같은 주제 연속 업로드 금지

## .env에 추가 필요
MAKE_YOUTUBE_WEBHOOK=웹훅URL
MAKE_INSTAGRAM_WEBHOOK=웹훅URL

## Claude Code 실행 코드
output/ 저장 후 자동 실행:
curl -X POST $MAKE_YOUTUBE_WEBHOOK \
  -H "Content-Type: application/json" \
  -d '{
    "title": "제목",
    "script": "스크립트",
    "platform": "youtube",
    "date": "날짜"
  }'

## 현실적 주의사항
- 영상 파일 없으면 유튜브 업로드 불가
- 인스타는 이미지/영상 파일 필요
- 텍스트만 있으면 스레드만 자동화 가능
- 영상 생성 AI 연결되면 완전 자동화 가능
