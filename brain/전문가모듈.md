# 전문가 모듈 (세무/법무/노무)

## 목표
전문가 영역 기본 정보 제공
단 항상 면책 문구 포함 + 전문가 상담 권고

## 세무 모듈

### 제공 가능한 것
- 사업자 등록 필요 시점 안내
  → 연 매출 1200만원 이상 시 통신판매업 신고 필요
- 세금 종류 기본 안내
  → 부가세: 연 매출 8000만원 이상 일반과세자
  → 소득세: 매년 5월 종합소득세 신고
- 경비 처리 가능 항목 안내
  → 서버비, API 비용, 장비 구입비 등
- 절세 기본 방법 안내

### 자동 트리거
매출 관련 질문 감지 시:
→ 기본 정보 제공
→ "정확한 세무는 세무사 상담 권장합니다" 필수 추가

## 법무 모듈

### 제공 가능한 것
- 이용약관 기본 템플릿 자동 생성
- 개인정보처리방침 기본 템플릿 자동 생성
- 계약서 기본 초안 작성
- 저작권 기본 체크리스트
- 플랫폼별 이용약관 요약

### 자동 트리거
법적 문서 요청 시:
→ 기본 템플릿 제공
→ "법적 효력은 법무사/변호사 검토 후 사용 권장" 필수 추가

## 노무 모듈

### 제공 가능한 것
- 프리랜서 계약 기본 안내
- 4대보험 기본 안내
- 근로계약서 기본 템플릿

## 면책 문구 (항상 자동 삽입)
세무 관련: "본 정보는 일반적인 안내이며 정확한 세무는 세무사와 상담하세요"
법무 관련: "본 내용은 참고용이며 법적 효력을 위해 전문가 검토가 필요합니다"
노무 관련: "정확한 노무 관련 사항은 노무사와 상담하세요"

## junho 전용 경고 시스템 (admin만)

### 세무 경고
매출 발생 시 자동 체크:
- 월 100만원 초과 → "⚠️ 사업자 등록 검토 필요"
- 연 1200만원 초과 → "🚨 통신판매업 신고 필수"
- 연 4800만원 초과 → "🚨 부가세 신고 필수"
- 연 8000만원 초과 → "🚨 일반과세자 전환 필요"

해결방안 자동 제시:
"세무사 연결 필요 → 크몽에서 세무사 찾기 또는
국세청 홈택스(hometax.go.kr) 에서 직접 신고 가능"

### 법무 경고
- 유저 100명 초과 → "⚠️ 이용약관 + 개인정보처리방침 필수"
- 결제 시작 → "⚠️ 전자상거래법 환불 정책 필수"
- 해외 유저 발생 → "⚠️ GDPR 개인정보처리방침 필요"

해결방안 자동 제시:
"이용약관 자동 생성 → skills/납품가이드생성.md 실행"
"개인정보처리방침 → 자동 초안 생성 가능"
