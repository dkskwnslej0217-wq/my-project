# 멀티모델 파이프라인

## 목표
Claude 혼자 다 하는 구조 → 3개 모델 분업
속도 3배, 비용 70% 절감, 품질 상승

## 분업 구조

```
[1단계] Gemini Flash — 트렌드 수집 + 초리서치
        ↓ 실패(429 쿼터초과) 시 → Groq로 폴백
[2단계] Groq (llama-3.3-70b-versatile) — 초안 생성
        ↓
[3단계] Claude — 품질 검수 + 최종 완성
```

## 폴백 규칙
- Gemini 429 → Groq로 1단계 대체 (자동)
- Groq 실패 → Claude 단독 처리
- 모든 실패 → brain/에러알림.md → 텔레그램 알림

## 각 모델 역할

### YouTube Data API (0단계 — 실데이터 수집)
- 한국 급상승 영상 TOP 10 직접 수집 (조회수·제목·채널)
- 키워드 검색으로 경쟁 영상 데이터 수집
- Tavily 간접 수집보다 정확한 실시간 데이터
- API: YOUTUBE_API_KEY (Gemini 같은 Google Cloud 프로젝트 키 사용 가능)

```bash
source .env
# 한국 급상승 TOP 10
curl -s "https://www.googleapis.com/youtube/v3/videos?part=snippet,statistics&chart=mostPopular&regionCode=KR&maxResults=10&key=$YOUTUBE_API_KEY" \
  | jq '[.items[] | {title:.snippet.title, views:.statistics.viewCount, channel:.snippet.channelTitle}]'

# 특정 키워드 검색
curl -s "https://www.googleapis.com/youtube/v3/search?part=snippet&q=$KEYWORD&order=viewCount&type=video&regionCode=KR&maxResults=5&key=$YOUTUBE_API_KEY" \
  | jq '[.items[] | {title:.snippet.title, channel:.snippet.channelTitle}]'
```

### Gemini Flash (1단계 — 리서치)
- YouTube 0단계 데이터 + Tavily 수집 트렌드 종합 요약
- 경쟁자 콘텐츠 분석
- 키워드 추출
- API: GEMINI_API_KEY

```bash
source .env
curl -s "https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash-lite:generateContent?key=$GEMINI_API_KEY" \
  -H "Content-Type: application/json" \
  -d "{
    \"contents\": [{
      \"parts\": [{\"text\": \"다음 트렌드를 분석해서 콘텐츠 키워드 5개 뽑아줘: $TREND_DATA\"}]
    }]
  }" |
```

### Groq / Llama3 (2단계 — 초안)
- Gemini가 뽑은 키워드로 초안 생성
- 훅 3가지 버전 생성
- 속도: 초당 500토큰 (Claude의 10배)
- API: GROQ_API_KEY

```bash
source .env
curl -s "https://api.groq.com/openai/v1/chat/completions" \
  -H "Authorization: Bearer $GROQ_API_KEY" \
  -H "Content-Type: application/json" \
  -d "{
    \"model\": \"llama3-8b-8192\",
    \"messages\": [{
      \"role\": \"user\",
      \"content\": \"키워드: $KEYWORDS\n\n스레드용 훅 3가지 버전 만들어줘. 각 50자 이내.\"
    }],
    \"max_tokens\": 500
  }" | 
```

### Claude (3단계 — 완성)
- Groq 초안 중 가장 좋은 것 선택
- 브랜드 톤 적용 (brain/정체성.md)
- 감정 흐름 체크 (brain/감정분석.md)
- 자기검증 (skills/자기검증.md)
- 최종 콘텐츠 완성

## run_core.md 연동 위치
기존 14단계 (콘텐츠 생성) 를 이 파이프라인으로 교체:
- 14a. YouTube API → 급상승 TOP 10 + 키워드 검색 수집
- 14b. Gemini Flash → 수집 데이터 요약 + 키워드 추출
- 14c. Groq → 훅 초안 3개
- 14d. Claude → 최종 완성
