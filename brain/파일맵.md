## 항상 로드 (필수 3개)
- brain/인증.md → 인증 확인 (실패 시 즉시 중단)
- brain/파일맵.md (현재 파일) → 필요 파일 선별 + 감사관 역할
- brain/Claude코드제한.md → 절대 금지 행동 정의

## 감사관 통합 체크리스트 (실행 전 자동 확인)
- [ ] 파일 충돌 없음 (같은 기능 중복 파일 없음)
- [ ] 규칙 일관성 (CLAUDE.md ↔ brain/ 파일 모순 없음)
- [ ] .env 필수 키 존재 (MASTER_PASSWORD, SUPABASE_URL, SUPABASE_KEY, USER_ID)
- [ ] LOCKOUT=false 확인
- [ ] API 상태 정상
- 이상 발견 시 → 자동 수정 또는 즉시 보고 후 중단
