# 영상 자동 생성 시스템

## 현재 상태
뼈대 완성 / 실제 연결 보류
영상 자동화 솔루션 확정 후 연결 예정

## 목표
스크립트 → 영상 → 플랫폼 업로드 완전 자동화

## 영상 생성 단계
1. run_core.md에서 스크립트 생성 완료
2. brain/영상생성.md 호출
3. 플랫폼 판단 (Instagram Reels / YouTube Shorts / Threads)
4. 영상 생성 엔진 선택 (아래 옵션 중 1개)
5. output/video/[날짜]_[플랫폼]_[주제].mp4 저장
6. brain/자동업로드.md → Make.com 웹훅 호출

## 영상 생성 엔진 옵션

- 비용: 거의 0원
- 퀄리티: 중간
- 자동화: 가능
- 연결 조건: ELEVENLABS_API_KEY .env에 추가 필요

- 자동화: 가능
- 연결 조건: HEYGEN_API_KEY .env에 추가 필요

- 월 $22~
- 퀄리티: 높음

## 플랫폼별 영상 스펙

### Instagram Reels
- 비율: 9:16 (세로)
- 해상도: 1080x1920
- 길이: 15~90초
- 포맷: MP4 (H.264)

### YouTube Shorts
- 비율: 9:16 (세로)
- 해상도: 1080x1920
- 길이: 60초 이하
- 포맷: MP4

### Threads
- 비율: 1:1 또는 9:16
- 해상도: 1080x1080 또는 1080x1920
- 길이: 5분 이하
- 포맷: MP4

## ffmpeg 자동 생성 코드 (옵션 A)

### 1. 음성 생성 (ElevenLabs)
```bash
curl -X POST "https://api.elevenlabs.io/v1/text-to-speech/$ELEVENLABS_VOICE_ID" \
  -H "xi-api-key: $ELEVENLABS_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"text": "[스크립트]", "model_id": "eleven_multilingual_v2"}' \
  --output output/video/temp_audio.mp3
```

### 2. 자막 이미지 생성 + 영상 합성 (ffmpeg)
```bash
ffmpeg -loop 1 -i output/video/bg.jpg \
  -i output/video/temp_audio.mp3 \
  -c:v libx264 -tune stillimage \
  -c:a aac -b:a 192k \
  -vf "scale=1080:1920" \
  -shortest output/video/[날짜]_[플랫폼].mp4
```

## 2단계: Pexels + ElevenLabs 자동화 (중간 퀄리티)

### 작동 구조
```
스크립트 생성
↓
ElevenLabs → 음성 파일 생성 (MP3)
↓
Pexels API → 주제 관련 배경 영상 클립 자동 검색 + 다운로드
↓
ffmpeg → 배경영상 + 음성 + 자막 합성 → MP4
↓
output/video/ 저장
↓
Make.com → 플랫폼 업로드
```

### Pexels 배경 영상 자동 검색
```bash
# Node.js로 처리 (python3 환경 문제로 대체)
curl -s -H "Authorization: $PEXELS_API_KEY" \
  "https://api.pexels.com/videos/search?query=[주제키워드]&per_page=3&orientation=portrait" \
  | node -e "let d='';process.stdin.on('data',c=>d+=c);process.stdin.on('end',()=>{const v=JSON.parse(d).videos[0].video_files;console.log(v.find(f=>f.quality==='hd').link);})"
```

### ElevenLabs 음성 생성
```bash
curl -X POST "https://api.elevenlabs.io/v1/text-to-speech/$ELEVENLABS_VOICE_ID" \
  -H "xi-api-key: $ELEVENLABS_API_KEY" \
  -H "Content-Type: application/json" \
  -d "{\"text\": \"[스크립트]\", \"model_id\": \"eleven_multilingual_v2\"}" \
  --output output/video/temp_audio.mp3
```

### ffmpeg 합성
```bash
ffmpeg -i output/video/temp_bg.mp4 \
  -i output/video/temp_audio.mp3 \
  -vf "scale=1080:1920:force_original_aspect_ratio=increase,crop=1080:1920,subtitles=output/video/temp_sub.srt" \
  -c:v libx264 -c:a aac \
  -shortest output/video/[날짜]_[플랫폼].mp4
```
