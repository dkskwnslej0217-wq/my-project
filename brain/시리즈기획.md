# 시리즈 기획 뇌

## 목표
1편~10편 시리즈 자동 설계
구독자가 다음 편 기다리는 구조 만들기

## 시리즈 구조 공식
1편: 문제 제기 (왜 이게 중요한가)
2편: 핵심 개념 (기초 설명)
3편: 실전 방법 1 (쉬운 것)
4편: 실전 방법 2 (심화)
5편: 실수 모음 (공감 유도)
6편: 도구 소개 (실용 정보)
7편: 비교 분석 (논쟁 유도)
8편: 성공 사례 (희망 유도)
9편: 심화 팁 (충성 구독자용)
10편: 총정리 (신규 유입용)

## 시리즈 생성 방법
주제 입력 시 "시리즈로 만들어줘" 명령 가능
→ 10편 제목 + 핵심 내용 자동 생성
→ memory/시리즈.md에 저장
→ 캘린더에 자동 배치

## 클리프행어 규칙
매 편 마지막에 다음 편 예고
"다음 편에서는 [구체적 내용] 알려드릴게요"
