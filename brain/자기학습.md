# 성과 기반 자기학습

## 목표
Supabase에 쌓인 점수 데이터를 분석해서
다음 콘텐츠 전략을 자동으로 개선한다

## 실행 순서
1. Supabase에서 최근 20개 기록 조회
   (date, topic, score_total, score_hook, score_trend)
2. score_total 7점 이상 → 성공 패턴으로 분류
3. score_total 5점 이하 → 실패 패턴으로 분류
4. 성공 패턴에서 공통점 추출
   - 주제 유형
   - 훅 문장 스타일
   - 트렌드 반영도
5. brain/패턴.md 자동 업데이트

## 패턴 업데이트 규칙
- 성공 패턴 3개 이상 쌓이면 → 우선순위 상단으로
- 실패 패턴 2번 이상 반복되면 → 자동 차단 목록 추가
- 매 5회 실행마다 자기학습 1회 실행
