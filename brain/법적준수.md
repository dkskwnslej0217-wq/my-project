# 전세계 법적 준수 시스템

## 목표
한국 + 미국 + EU + 전세계 법 자동 모니터링
각 SNS 플랫폼 정책 준수 자동 체크

## 국가별 법적 체크

### 한국
- 저작권법: 타인 콘텐츠 무단 사용 금지
- 정보통신망법: 개인정보 수집 시 동의 필수
- 표시광고법: 광고/협찬 반드시 표시
- 전자상거래법: 디지털 상품 판매 시 환불 정책 필수
- 금융소비자보호법: 투자/금융 정보 제공 시 면책 문구 필수

### 미국
- DMCA: 저작권 침해 콘텐츠 즉시 삭제
- FTC 가이드라인: 광고/협찬 "#AD" 또는 "#Sponsored" 표시 필수
- CAN-SPAM: 이메일 마케팅 시 수신 거부 옵션 필수
- CCPA: 캘리포니아 사용자 개인정보 보호

### EU
- GDPR: 유럽 사용자 개인정보 수집 시 명시적 동의 필수
- 쿠키 정책: 웹사이트 쿠키 사용 동의 필수
- DSA: 플랫폼 콘텐츠 책임 강화

### 전세계 공통
- 아동 관련 콘텐츠: COPPA (13세 미만 데이터 수집 금지)
- 증오 발언: 모든 플랫폼 공통 금지
- 허위 정보: 팩트 체크 필수
- 개인정보: 수집 최소화 원칙

## SNS 플랫폼별 정책

### 유튜브
- 광고 수익: 1000명 구독 + 4000시간 시청 필요
- 저작권: Content ID 시스템 자동 감지
- 스팸: 반복 콘텐츠 금지
- 협찬: 유튜브 자체 협찬 공개 기능 사용 필수

### 인스타그램
- 광고: "#광고" 또는 "#AD" 필수
- 자동화: 봇 사용 금지 (Make.com 공식 API만 허용)
- 저작권: 음악/이미지 라이선스 확인 필수
- 성인 콘텐츠: 18+ 제한 설정 필수

### 스레드
- 메타 커뮤니티 가이드라인 준수
- 자동 게시: API 사용 시 제한 있음
- 스팸: 동일 내용 반복 금지

### 네이버 블로그
- 애드포스트: 광고성 콘텐츠 표시 필수
- 저작권: 이미지 출처 표시 필수
- 의료/법률/금융: 전문가 상담 권고 문구 필수

### 트위터/X
- 자동화: API 사용 시 1500 트윗/월 제한 (무료)
- 광고: 협찬 표시 필수
- 스팸: 동일 내용 반복 계정 정지 위험

## 콘텐츠 자동 체크 항목
매 콘텐츠 생성 후 자기검증에 추가:
- [ ] 광고/협찬이면 "#광고" "#AD" 표시했는가?
- [ ] 타인 저작물 무단 사용 없는가?
- [ ] 허위/과장 정보 없는가?
- [ ] 개인정보 포함 없는가?
- [ ] 아동 관련 부적절 콘텐츠 없는가?
- [ ] 증오 발언 없는가?
- [ ] 금융/투자 정보면 면책 문구 있는가?

## 면책 문구 자동 삽입
플랫폼별 자동 삽입:
- 금융/투자 콘텐츠 → "본 콘텐츠는 투자 권유가 아닙니다"
- 의료 콘텐츠 → "전문가 상담을 권장합니다"
- 광고 콘텐츠 → "#광고 #AD"
