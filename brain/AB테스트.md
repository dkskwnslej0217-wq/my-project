# A/B 실험 시스템

## 목표
훅/포맷/길이/주제방향 자동 실험해서
최고 성과 조합 찾기

## 실험 항목

### 훅 실험
A: 반전형 "X → Y"
B: 숫자형 "3가지", "5분 만에"
C: 질문형 "왜?", "어떻게?"
D: 공감형 "저도 몰랐어요"

### 포맷 실험
A: 짧고 강렬 (3문장 이내)
B: 스토리텔링 (기승전결)
C: 리스트형 (1,2,3 순서)
D: 문답형 (질문→답변)

### 길이 실험
A: 초단형 (30초 이내)
B: 단형 (1분)
C: 중형 (3분)

## 실험 방법
1. 매 콘텐츠 생성 시 각 항목별 랜덤 조합 선택
2. 조합 Supabase에 기록
3. score_total과 함께 저장
4. 20개 쌓이면 최고 성과 조합 자동 분석
5. brain/개인화.md에 최강 조합 반영
